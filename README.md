# FrameForge

Universal Cinematic Frame Smoothing Tool for Low-End PCs.

## Project Structure

- `launcher/`: Tauri-based desktop application (UI and backend).
- `runtime/`: C++ DirectX 11 hook and interpolation engine.
- `shared/`: Shared headers and communication protocols.
- `external/`: Third-party libraries (MinHook, Dear ImGui, etc.).
- `profiles/`: Per-game configuration profiles.
- `docs/`: Documentation and research.
- `logs/`: Runtime and launcher logs.
